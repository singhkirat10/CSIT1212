XML

What is XML?

XML stands for eXtensible Markup Language. It is a markup language and a file-format that is designed for storing and transmitting data. There are no predefined tags to use, instead this tool allows one to define their own tags based on the needs. Since, the format of XML is standardized, it helps transmit XML across systems, platforms, locally or over the internet, and the recipient can parse data due to standardized rules/syntax

How is XML used?

XML is used for many aspects of web-development, including storing and transferring data across systems and applications. XML is only data, it does not have any information about how to display that data. Because of the standard format of XML, it can be parsed and interpreted by all types of programming languages and applications.
Some of the uses of XML are:

  • Data Transfer
  • Formatting documents
  • Web Applications
  • Data type to support programming language

JavaScript

What is JavaScript?

JavaScript is a programming or scripting language for the webpages. JavaScript can dynamically update content, control multimedia, animate images, and provide interoperability across different browsers and devices.

How is JavaScript related to HTML and CSS?

JavaScript is either embedded directly into a webpage or reference from via a separate .js file. The browser runs this script along side HTML and CSS code, that the user is trying to open in browser; thus creating a functional, dynamic page.

We use HTML to give structure and purpose to our content. CSS adds style rules to the content for display, for example background colors, fonts, layouts. And JavaScript add functionality and dynamic behavior that can respond to user interactions on a webpage

Common JavaScript elements

Some common elements you will come across in JavaScript are: Variables (var), Objects, functions, events.
